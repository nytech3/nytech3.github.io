Structs of Sds

There are five struct that sds has. They are sdshdr5,sdshdr8,sdshdr16,sdshdr32,sdshdr64, you can find the structs in sds.h.

DEBUG问题

开始之前写一下自己用mac的CLION DEBUG Redis 源码时遇到的问题，主要是因为自己对于编译C语言不是很了解，所以浪费了一些时间。
想要进行debug，需要禁止下编译优化，即将 src -> Makefile中的OPTIMIZATION?=-O3 改成 OPTIMIZATION?=-O0， 再将REDIS_LD=$(QUIET_LINK)$(CC) $(FINAL_LDFLAGS)
改成REDIS_LD=$(QUIET_LINK)$(CC) $(FINAL_LDFLAGS) $(OPTIMIZATION)

SDSHDR

As you can see, the type of flags is char. It presents sds type by 3 lsb, and presents sds len by 5 msb that it is the biggest different with other 4 types.
i.e: 01000000 means that len is 01000(2^4=16), and type is 000(SDS_TYPE_5 is defined 0 in redis)
